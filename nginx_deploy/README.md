# nginx Deploy Cookbook
==============
A custom deployment of the nginx web server including some baseline configuration.

Requirements
------------
- Chef (obviously)
- Ruby
- Berkshelf

This is specifically to be used in an Amazon AWS / OpsWorks environment.
